# Shared LaTeX Config

Most of my LaTeX writing so far has been confined to my school "mono"repo,
[SpeedSchool](https://github.com/Joe-Downs/SpeedSchool). In working on my
[Master's Thesis](https://github.com/Joe-Downs/thesis), I wanted the same
configurations I had, but didn't want to duplicate them. To remedy this, I cut
out the relevant commits from the SpeedSchool repo and brought them here.

The `-cfg.tex` files are my own work, but the `ulthesis.cls` is taken from
UofL's [Physics and Astronomy
department](https://prancer.physics.louisville.edu/ulthesis/) thanks to work by
Dr. John Kielkopf and others.
